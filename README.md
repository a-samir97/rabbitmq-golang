# RabbitMQ and Golang 

Simple Pub/Sub using RabbitMQ and Golang

## Tools 
- Golang
- RabbitMQ (Pub/Sub)

## how to run project ?

- Assuming you have rabbitmq server is running 
- run this commands 
    - go run publisher.go
    - go run consumer.go
    
## Objectives

- to learn pub/sub architecture
